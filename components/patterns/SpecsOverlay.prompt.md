Full-page spec sheet that reads the site's tokens at runtime and lays them out: type scale with live samples, color swatches with resolved values, spacing bars, faces and motion. The quiet engineer flex.

```jsx
<SpecsOverlay />
```

Mount once per page. Opens on the `.` key (ignored while typing) or a `pf-specs-open` window event (footer "specs" link and the command menu dispatch it). Esc closes.
