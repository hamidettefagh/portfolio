Inline text link with hairline underline; hover darkens the underline to currentColor.

```jsx
<TextLink href="mailto:x">hamid.ettefagh@gmail.com</TextLink>
<TextLink href="https://linkedin.com/in/..." external>LinkedIn</TextLink>
```

`accent` colors it burnt orange; `external` opens a new tab and adds the northeast arrow (the system's external-link glyph).
