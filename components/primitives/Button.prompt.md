Pill-shaped button; use exactly one primary (accent-filled) per view, secondary outline for the rest.

```jsx
<Button href="#work" arrow>See the work</Button>
<Button variant="secondary" size="sm">Read the case</Button>
```

Variants: `primary` (burnt-orange fill), `secondary` (hairline outline), `ghost` (bare text). Sizes `sm`/`md`. `arrow` adds a &rarr; that nudges right on hover. Pass `href` to render as a link.
